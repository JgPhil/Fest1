Compétences évaluées

  • Expliquer le fonctionnement du DOM et des méthodes JavaScript

  • Question 1

    Pour inclure du code JavaScript dans une page web, on utilise une balise de type :

    • <link src="..." />

    • <link href="..." />

    • <script src="..." />

    • <script href="..." />

    Une balise de type <link href="..." /> sert à inclure une feuille de style CSS. Avec une balise <script>, on doit utiliser l'attribut src et non href pour indiquer l'emplacement du fichier JavaScript.

  • Question 2

    JavaScript est le seul langage de programmation inclus dans l'ensemble des navigateurs web.

    • Vrai

    • Faux

    JavaScript a été créé spécifiquement pour être utilisé avec les pages web. Il s’est imposé comme le langage de programmation standard du Web, et tous les navigateurs du marché le prennent en charge.

  • Question 3

    Trouvez les affirmations vraies à propos du DOM.

    Attention, plusieurs réponses sont possibles.

    • Le DOM est une représentation de la structure d’une page web sous la forme d’une hiérarchie de noeuds.

    • Le DOM est spécifique à chaque navigateur.

    • Le DOM est une API qui permet d’interagir avec la page web.

    • Chaque noeud du DOM correspond en JavaScript à un tableau.

    Le DOM est à la fois une représentation hiérarchique de la structure d’une page web et une interface de programmation (API) pour accéder à cette structure. Le DOM a été standardisé par le W3C. Chaque noeud du DOM correspond à un objet JavaScript.

  • Question 4

    Soit le code HTML ci-dessous. Quelle instruction JavaScript permet d’accéder à l’élément identifié par “cible” ?

    <body>

        <ul id="courses">

            <li>Beurre</li>

            <li id="cible">Oeufs</li>

            <li>Lait</li>

        </ul>

    </body>

    • document.body.childNodes[0].childNodes[1];

    • document.body.childNodes[1].childNodes[3];

    • document.body.childNodes[1].childNodes[3].childNodes[0];

    • document.body.childNodes[2];

    L’élément identifié par “cible” est, en comptant les noeuds textuels “vides”, le 4e enfant de l’élément <ul>, qui est lui-même le 2e enfant de l’élément <body>. Les indices des listes de noeuds enfants commencent toujours à 0.

  • Question 5

    Soit le code HTML ci-dessous. Quelles instructions JavaScript permettent d’accéder à l’élément identifié par “cible” ?

    <body>

        <ul id="courses">

            <li>Beurre</li>

            <li id="cible">Oeufs</li>

            <li>Lait</li>

        </ul>

    </body>
    Attention, plusieurs réponses sont possibles.

    • document.getElementById("cible");

    • document.getElementById("#cible");

    • document.getElementsByTagName("li")[1];

    • document.getElementsByClassName("li")[1];

    • document.querySelector("cible");

    • document.querySelector("#cible");

    La méthode getElementById prend en paramètre un identifiant, contrairement à querySelector qui prend en paramètre un sélecteur CSS. La méthode getElementsByTagName renvoie une liste d’objets commençant à l’indice 0. Comme aucune classe n’est définie, la méthode getElementsByClassName n’est pas utilisable ici.

  • Question 6

    Comment faire pour accéder à l’URL d’un objet JavaScript nommé lien et représentant un élément <a> du DOM ?

    • lien.href

    • lien.src

    • lien.url

    • lien.value

    Seule la propriété href permet d’accéder à l’URL. Les propriétés src et url n’existent pas. La propriété value n’existe pas pour les éléments <a>.

  • Question 7

    Soit le code HTML ci-dessous.

    <body>

        <ul id="courses">

            <li>Beurre</li>

            <li id="cible">Oeufs</li>

            <li>Lait</li>

        </ul>

    </body>

    Que fait le code JavaScript associé ?

    var farine = document.createElement("li");

    farine.id = "Farine";

    farine.textContent = "farine";

    document.getElementById("courses").appendChild(farine);

    • Il ajoute un élément affichant le texte “Farine” au début de la liste.

    • Il ajoute un élément affichant le texte “Farine” à la fin de la liste.

    • Il ajoute un élément affichant le texte “farine” au début de la liste.

    • Il ajoute un élément affichant le texte “farine” à la fin de la liste.

    La méthode appendChild ajoute un élément comme dernier fils. Le texte de l’élément est défini par la propriété textContent.

  • Question 8

    Outre appendChild, quelles méthodes JavaScript permettent d’ajouter de nouveaux éléments à une page web ?

    Attention, plusieurs réponses sont possibles.

    • insertBefore

    • insertAfter

    • insertHTML

    • insertAdjacentHTML

    Les méthodes insertAfter et insertHTML n’existent pas.

  • Question 9

    Quelle instruction permet de donner la valeur “none” à la propriété CSS list-style d’un objet JavaScript nommé liste et représentant un élément <ul> du DOM ?

    • liste.style.list-style = “none”;

    • liste.style.listStyle = “none”;

    • liste.list-style = “none”;

    • liste.listStyle = “none”;

    La propriété style d’un élément du DOM permet de modifier ses propriétés CSS. Les propriétés CSS composées comme list-style  s’écrivent en JavaScript avec la notation camelCase.

  • Question 10

    Comment faire pour accéder à la propriété CSS display d’un objet JavaScript nommé elt et représentant un élément du DOM ?

    • elt.getComputedStyle(“display”);

    • elt.getComputedStyle().display;

    • getComputedStyle(elt).display;

    • elt.style.display;

    La propriété style d’un élément du DOM ne permet pas d’accéder au style complet de l’élément : il faut utiliser getComputedStyle, qui prend en paramètre un noeud du DOM et renvoie un objet représentant son style. Il s’agit d’une fonction et non d’une méthode.
